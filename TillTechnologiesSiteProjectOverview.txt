### Core Technologies
- **Frontend Framework**: React.js (with Next.js for SSR/SSG capabilities)

- Page transition animations
- Parallax scrolling (optional)

## ğŸš€ Setup Instructions for VS Code

### Step 1: Initialize Project

```bash
# Create new Next.js app with TypeScript
npx create-next-app@latest tilltechnologies-portfolio --typescript --tailwind --app

# Navigate to project directory
cd tilltechnologies-portfolio

# Open in VS Code
code .
```

### Step 2: Install Dependencies

```bash
# Core dependencies
npm install framer-motion react-icons

# Map integration
npm install leaflet react-leaflet
npm install -D @types/leaflet

# Typewriter effect
npm install react-type-animation

# Image gallery
npm install react-photo-album yet-another-react-lightbox

# Date utilities
npm install date-fns

# Form handling
npm install react-hook-form

# API calls
npm install axios swr
```

### Step 3: Configure Environment Variables

Create `.env.local` file:
```env
# Strava API
NEXT_PUBLIC_STRAVA_CLIENT_ID=your_client_id
STRAVA_CLIENT_SECRET=your_client_secret
STRAVA_REFRESH_TOKEN=your_refresh_token

# GitHub (if using authenticated API)
GITHUB_TOKEN=your_github_token

# Whoop API (if available)
WHOOP_API_KEY=your_api_key

# EmailJS (for contact form)
NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key

# Google Analytics (optional)
NEXT_PUBLIC_GA_MEASUREMENT_ID=your_measurement_id
```

### Step 4: Configure Tailwind CSS

Update `tailwind.config.js`:
```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './src/pages/**/*.{js,ts,jsx,tsx,mdx}',
    './src/components/**/*.{js,ts,jsx,tsx,mdx}',
    './src/app/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {
      colors: {
        text: '#e9f1e9',
        background: '#020302',
        primary: '#42ba40',
        secondary: '#534a64',
        accent: '#1eae19',
      },
      fontFamily: {
        pixel: ['"Press Start 2P"', 'cursive'],
        lato: ['Lato', 'sans-serif'],
      },
    },
  },
  plugins: [],
}
```

### Step 5: Set Up Global Styles

Update `src/app/globals.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=Lato:wght@300;400;700;900&family=Press+Start+2P&display=swap');

@tailwind base;
@tailwind components;
@tailwind utilities;

:root {
  --text: #e9f1e9;
  --background: #020302;
  --primary: #42ba40;
  --secondary: #534a64;
  --accent: #1eae19;
}

body {
  color: var(--text);
  background: var(--background);
  font-family: 'Lato', sans-serif;
}

/* Smooth scrolling */
html {
  scroll-behavior: smooth;
}

/* Custom scrollbar */
::-webkit-scrollbar {
  width: 10px;
}

::-webkit-scrollbar-track {
  background: var(--background);
}

::-webkit-scrollbar-thumb {
  background: var(--primary);
  border-radius: 5px;
}

::-webkit-scrollbar-thumb:hover {
  background: var(--accent);
}
```

### Step 6: VS Code Extensions (Recommended)

Install these extensions for better development experience:
- **ES7+ React/Redux/React-Native snippets**
- **Tailwind CSS IntelliSense**
- **Prettier - Code formatter**
- **ESLint**
- **Auto Rename Tag**
- **Path Intellisense**
- **GitLens**

### Step 7: Create Initial Component Structure

```bash
# Create component directories
mkdir -p src/components/{layout,home,about,portfolio,skills,health,interests,travels,contact,common}

# Create page directories (if using pages router)
mkdir -p src/pages

# Create data directory
mkdir -p src/data

# Create lib directory for API integrations
mkdir -p src/lib

# Create public assets directories
mkdir -p public/{images/{projects,photos,profile},resume,fonts}
```

### Performance

### Accessibility
- Semantic HTML
- ARIA labels
- Keyboard navigation
- Color contrast ratios
- Alt text for images

### SEO
- Meta tags for each page
- Open Graph tags
- Structured data (JSON-LD)
- Sitemap generation
- robots.txt

### Security
- Environment variables for API keys
- Rate limiting on API routes
- Input validation
- Content Security Policy headers
